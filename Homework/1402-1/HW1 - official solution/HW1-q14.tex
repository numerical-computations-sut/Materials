می‌دانیم که فرمول کلی سری مکلورن به شکل 

$$\frac{1}{1 + x} = 1  - x + x^{2} - x^{3} + ...$$

می‌باشد. بنابراین، سری مکلورن $\frac{1}{2 + x}$ را پیدا کنید.

\begin{comment}
\end{comment}
پاسخ:


    $$\frac{1}{2 + x} = \frac{1}{2 (1 + \frac{x}{2})}$$
    $$= 2 (1 - \frac{x}{2} + \frac{x^{2}}{4} - \frac{x^{3}}{8}) + ...$$
