سری تیلور توابع زیر را حول $x_{0}$ حساب کنید:

\begin{itemize}
    \item $$f(x) = e^{-6x}, x_{0} = 4$$
    \item $$f(x) = \ln(3 + 4x), x_{0} = 0$$
    \item $$f(x) = \frac{7}{x^{4}}, x_{0} = -3$$
\end{itemize}
